# Ed's Power BI Learning Journey

This repository documents my progress in mastering **Power BI** through **Microsoft Learn** and other resources.  
It serves as a portfolio of my skills in **data visualization, analytics, and business intelligence**, aligned with my IT career transition.

---

## Learning Goals
- Understand the fundamentals of **Power BI Desktop** and **Power BI Service**.
- Learn to connect, transform, and model data from multiple sources.
- Build interactive dashboards and reports with professional design.
- Apply **DAX (Data Analysis Expressions)** for advanced calculations.
- Share insights effectively across GitHub and LinkedIn.

- `/notes` → Markdown notes summarizing modules (Data Modeling.md, DAX.md, Visualization.md).
